# 🏦 RiskLens

## AI-Powered IFRS 9 Credit Risk Assessment & Expected Credit Loss (ECL) Decision Support System

RiskLens is an end-to-end machine learning application that assists financial institutions in evaluating borrower credit risk using **IFRS 9 Expected Credit Loss (ECL)** methodology. The system combines predictive machine learning, explainable AI, and banking business rules to support transparent and data-driven lending decisions.

---

## 🚀 Key Features

- 🔍 Customer search with banking-style Applicant IDs
- 🤖 LightGBM-based Credit Risk Prediction (P1–P4)
- 📊 IFRS 9 Expected Credit Loss (ECL) Calculation
- 📈 PD, LGD, EAD & Stage Classification
- 💳 Credit Decision Score Engine
- 🔄 Interactive What-If Analysis
- 🧠 SHAP Explainability (Global & Individual)
- 📉 Comprehensive Model Performance Dashboard
- 🏦 Professional Streamlit Banking Interface
